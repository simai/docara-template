# Docara

Docara - фреймворк документации для проектов, которые хранят документацию рядом с кодом.

## Преимущества

### Простое содержание

- Пишите страницы в Markdown.
- Храните документацию в одном репозитории с проектом.
- Проверяйте изменения через pull request.

### Предсказуемая структура

- Исходные файлы лежат в `source/docs`.
- Навигация настраивается через `.settings.php`.
- Результат собирается в статический сайт.

### Локальный просмотр

- Запускайте сайт в Codespaces.
- Меняйте страницы и быстро пересобирайте проект.
- Делитесь предпросмотром через forwarded ports.
