# О проекте

Этот шаблон - минимальный проект документации на Docara.

Используйте его, когда нужно быстро начать документационный сайт без ручной подготовки структуры. В шаблон входят:

- Markdown-страницы.
- Левое меню навигации.
- Навигация по статье справа.
- Сборка ассетов через Laravel Mix.
- Конфигурация GitHub Codespaces.

Замените эту страницу описанием своего проекта, аудитории и правил участия.
