# Docara

Docara is a documentation framework for projects that keep docs close to code.

## Advantages

### Simple Content

- Write pages in Markdown.
- Keep documentation in the same repository as the project.
- Review documentation changes through pull requests.

### Predictable Structure

- Store source files in `source/docs`.
- Configure navigation with `.settings.php`.
- Build static output for hosting anywhere.

### Local Preview

- Run the site in Codespaces.
- Edit pages and rebuild quickly.
- Share the result through forwarded ports.
