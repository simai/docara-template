# Documentation Template

This is a small Docara project for GitHub Codespaces. It contains a few Markdown pages, a nested section, and enough configuration to start writing documentation immediately.

## Start Here

- [About the project](about.md)
- [Platform page](page.md)
- [Project structure](structure.md)
- [Sub-section](sub-section/index.md)

## Edit Content

Open files in `source/docs/en` and change this text. Rebuild the site with:

```bash
yarn build
```

For a local preview, run:

```bash
yarn start
```
