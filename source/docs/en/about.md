# About

This template is a minimal documentation project built with Docara.

Use it when you want to start a documentation site without preparing the project structure by hand. The template includes:

- Markdown content pages.
- A left navigation menu.
- A right article navigation.
- Asset compilation through Laravel Mix.
- GitHub Codespaces configuration.

Update this page with your own project overview, audience, and contribution rules.
